# ICTラウンドアプリ「めぐる君」 取扱説明書

**文書番号:** ICT-UG-001  
**対象者:** ICTメンバー・感染対策ラウンド実施者  
**作成日:** 2026年4月18日

---

## 1. アプリ概要

### 1.1 目的
本アプリは、病院内の感染対策ラウンドを効率的に実施・記録するためのツールです。チェックリスト評価、写真記録、総評入力、Word 報告書出力を 1 つの画面遷移で行えます。

### 1.2 アプリ名称
**ICTラウンドアプリ「めぐる君」**  
開始画面や外観設定では、キャラクターアイコンとして「らんちゃん」「めぐる君」を切り替えできます。

### 1.3 動作環境

| 項目 | 要件 |
|------|------|
| 対応端末 | スマートフォン、タブレット、PC |
| 対応OS | iOS、Android、Windows、macOS |
| 対応ブラウザ | Safari、Chrome、Edge の比較的新しい版 |
| ネットワーク | 初回アクセス時と更新取得時に必要 |

> Service Worker によりアプリ本体はキャッシュされます。初回読み込み後は、同じ端末・同じブラウザで再訪した際にオフラインでも起動できる場合があります。

## 2. ホーム画面への追加

アプリは PWA として配布されており、ホーム画面やアプリ一覧に追加して起動できます。

### iPhone / iPad の場合（Safari）
1. Safari でアプリの URL を開く
2. 共有ボタンをタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

### Android の場合（Chrome）
1. Chrome でアプリの URL を開く
2. メニューを開く
3. 「ホーム画面に追加」または「アプリをインストール」を選択
4. 画面の案内に従って追加

## 3. 操作手順

### 3.1 ラウンド開始

1. アプリを起動すると開始画面が表示されます
2. 使用するチェックリストを選択します
3. **担当者名**を入力します（必須）
4. 必要に応じて **病棟名** を入力します（任意）
5. 「ラウンド開始」をタップします
6. 実施日時が自動入力され、メイン画面が開きます

---

### 3.2 チェックリストの選択・取り込み

開始画面では、使用するチェックリストを切り替えられます。

- 初回起動時は **標準チェックリスト** が自動で登録されています
- 標準チェックリストは **9カテゴリ・22項目** です
- `取り込む` ボタンから独自チェックリストを追加できます

**取り込み手順**

1. 開始画面の `取り込む` をタップ
2. 必要ならチェックリスト名を入力
3. `.csv` / `.xls` / `.xlsx` ファイルを選択
4. プレビュー内容を確認
5. `保存して適用` をタップ

**取り込みファイルの形式**

- 1列目: カテゴリ名
- 2列目: 点検項目
- 見出し行は省略可
- Excel は先頭シートのみ読み込みます

---

### 3.3 チェックリスト評価

ラウンド開始後は `チェック` タブが開きます。

**評価方法**

| 評価 | 意味 |
|------|------|
| **A** | 適切・問題なし |
| **B** | 概ね良好・改善余地あり |
| **C** | 要改善 |

- 各項目の A / B / C ボタンをタップして評価します
- 画面上部とカテゴリごとに進捗が表示されます
- カテゴリは開閉できます

---

### 3.4 写真の撮影・追加

写真は `写真` タブから追加できます。チェック項目に紐付く写真と、汎用写真の両方に対応しています。

**汎用写真を追加する場合**

1. `写真` タブを開く
2. `写真を追加` をタップ
3. カメラ撮影または写真選択を行う
4. 必要に応じてコメントを入力
5. `追加する` をタップ

**項目に関連する写真として追加する場合**

1. 該当項目から写真追加導線を開く
2. 写真を選択または撮影する
3. 必要に応じてコメントを入力
4. `追加する` をタップ

**写真に関する仕様**

- 取り込み可能なのは 10MB 以下の画像ファイルです
- 画像はブラウザ内で JPEG に圧縮して保存されます
- 追加済み写真は `写真` タブから削除できます

---

### 3.5 総評の入力

1. `総評` タブを開きます
2. テキストエリアに総評を入力します
3. 必要に応じて以下を使います
   - `・ 箇条書き` ボタン: 箇条書き記号を追加
   - マイクボタン: 音声入力を開始 / 停止

> 音声入力は対応ブラウザのみ利用できます。非対応環境ではボタン操作時に案内が表示されます。

---

### 3.6 保存済みラウンドの保存・再開

本アプリでは、途中経過を手動保存できます。

**保存する方法**

1. メイン画面上部の `保存` をタップ
2. 保存が成功すると `保存済み` 表示になります

**再開する方法**

1. 開始画面で `保存済みラウンドを開く` をタップ
2. 一覧から対象ラウンドを選んで `開く` をタップ

**削除する方法**

1. 保存済みラウンド一覧を開く
2. 対象ラウンドの `削除` をタップ

---

### 3.7 レポートプレビュー・出力

1. 画面下部の `レポート` をタップ
2. プレビュー画面で内容を確認します
3. 右上のボタンから出力します

出力方法は端末により異なります。

- **共有**: Web Share API で `.docx` ファイルを共有
- **Word出力**: `.docx` ファイルを保存

報告書には以下が含まれます。

- 担当者
- 病棟
- 実施日時
- チェックリスト評価
- 写真記録
- 総評

## 5. 注意事項

### 5.1 データ保存について

- 画面上で入力中の内容は自動保存されません
- 途中経過を残すには、必ず `保存` ボタンを押してください
- 保存済みラウンド、チェックリスト、テーマ、アイコンは端末ブラウザの `localStorage` に保存されます
- ブラウザのデータ削除やサイトデータ消去を行うと保存内容は削除されます

### 5.2 写真について

- 写真に患者の個人情報が写り込まないよう注意してください
- 写真は外部サーバーへ送信されません
- 保存済みラウンドに含めた写真は端末ブラウザ内に保持されます

### 5.3 音声入力について

- 音声認識機能はブラウザ依存です
- 利用時は周囲の会話や個人情報の発話に注意してください

## 6. よくある質問（FAQ）

**Q. 入力途中で閉じても続きから使えますか？**  
A. 自動保存はされません。続きから使うには、事前に `保存` ボタンで保存済みラウンドとして残してください。

**Q. 保存済みラウンドはどこに保存されますか？**  
A. この端末・このブラウザの `localStorage` に保存されます。他端末とは共有されません。

**Q. チェックリストを病棟ごとに分けられますか？**  
A. できます。CSV または Excel を取り込み、開始画面で使うチェックリストを選択してください。

**Q. 写真が追加できません。**  
A. ブラウザのカメラまたは写真ライブラリへのアクセス許可、またはファイルサイズ上限 10MB を確認してください。

**Q. 音声入力が使えません。**  
A. 端末またはブラウザが `SpeechRecognition` に対応していない可能性があります。その場合は手入力してください。

**Q. 出力形式は PDF ですか？**  
A. いいえ。出力形式は `.docx` です。

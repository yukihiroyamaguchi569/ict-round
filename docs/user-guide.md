# ICTラウンドアプリ「らんちゃん」 取扱説明書

**文書番号:** ICT-UG-001  
**対象者:** ICTメンバー・感染対策ラウンド実施者  
**作成日:** 2026年4月  

---

## 1. アプリ概要

### 1.1 目的
本アプリは、病院内の感染対策ラウンドを効率的に実施・記録するためのツールです。チェックリストへの評価入力、現場写真の記録、総評の作成、そして報告書のWord出力までを、スマートフォン・タブレット上で一貫して行えます。

### 1.2 アプリ名称
**ICTラウンドアプリ「らんちゃん」**（正式名称：ICN感染対策ラウンド）

### 1.3 動作環境

| 項目 | 要件 |
|------|------|
| 対応端末 | スマートフォン、タブレット、PC |
| 対応OS | iOS、Android、Windows、macOS |
| 対応ブラウザ | Safari（iOS）、Chrome（Android/PC）、Edge（PC）の最新版 |
| ネットワーク | 初回アクセス時のみ必要（以降はオフラインで動作可能） |

---

## 2. ホーム画面へのインストール（推奨）

アプリをホーム画面にインストールすると、ネイティブアプリのように素早く起動できます。

### iPhoneの場合（Safari）
1. SafariでアプリのURLを開く
2. 画面下部の共有ボタン（□に↑のアイコン）をタップ
3. 「ホーム画面に追加」を選択
4. 右上の「追加」をタップ

### Androidの場合（Chrome）
1. ChromeでアプリのURLを開く
2. 画面右上のメニュー（⋮）をタップ
3. 「ホーム画面に追加」または「アプリをインストール」を選択
4. 「追加」または「インストール」をタップ

> インストール後は、ホーム画面のアイコンからオフラインでも起動できます。

---

## 3. 操作手順

### 3.1 ラウンド開始

1. アプリを起動すると「ラウンド開始」画面が表示されます
2. **担当者名**を入力します（必須）
3. **病棟名**を入力します（任意）
4. 「ラウンド開始」ボタンをタップします
5. ラウンド開始日時が自動記録されます

---

### 3.2 チェックリスト評価

ラウンド開始後、「チェックリスト」タブが開きます。

**チェック対象：9カテゴリ・計22項目**

| カテゴリ | 項目数 |
|---------|--------|
| 手指衛生 | 2項目 |
| 水回り | 3項目 |
| 薬品 | 2項目 |
| リネン等 | 2項目 |
| 処置室 | 3項目 |
| PPE | 3項目 |
| 日常清掃 | 2項目 |
| デバイス関連 | 2項目 |
| 汚物室・トイレ | 3項目 |

**評価方法**

各項目を以下の3段階で評価します：

| 評価 | 意味 | 表示色 |
|------|------|--------|
| **A** | 適切・問題なし | 緑 |
| **B** | 概ね良好・軽微な改善が望ましい | 黄 |
| **C** | 要改善・指導が必要 | 赤 |

- 各項目のA/B/Cボタンをタップして評価を選択します
- タップ済みの評価を再タップすると選択解除されます
- 画面上部に評価済み件数の進捗が表示されます

---

### 3.3 写真の撮影・追加

**チェック項目に紐付けて写真を追加する場合**
1. チェックリスト画面で、写真を追加したい項目の📷アイコンをタップ
2. 「タップして撮影 / 選択」をタップ
3. カメラで撮影するか、ギャラリーから画像を選択します
4. 必要に応じてコメントを入力
5. 「追加する」ボタンをタップ

**一般写真として追加する場合**
1. 画面下部の「写真」タブをタップ
2. 「写真を追加」をタップして同様の手順で追加

> 写真は自動的に圧縮（最大幅1200px）されます。1枚あたりのファイルサイズは概ね300KB以下になります。

---

### 3.4 総評の入力

1. 画面下部の「総評」タブをタップ
2. テキストエリアに総評を入力します
3. 以下の補助機能が使用できます：
   - **「・ 箇条書き」ボタン** — 箇条書き記号を自動挿入
   - **マイクボタン** — 音声入力（日本語対応）

> 音声入力はSafari（iOS）およびChrome（Android）で利用できます。利用できない端末では手入力してください。

---

### 3.5 レポートプレビュー・出力

1. チェックリスト/写真/総評の入力が完了したら、「レポート」ボタンをタップ
2. 報告書のプレビューが表示されます（担当者名、病棟名、実施日時、チェック結果、写真、総評）
3. 右上のボタンから出力します：
   - **「Word出力」**（PC・一部Android） — `.docx` ファイルをダウンロード
   - **「共有」**（iOS・一部Android） — OS標準の共有シートが開き、メール・Teams等への送信や保存が可能

---

## 4. テーマの切替

ホーム画面右上のアイコンからテーマを切り替えられます。

| テーマ名 | 特徴 |
|----------|------|
| やさしい | 温かみのある、親しみやすいデザイン |
| ミニマル | モダンで洗練されたデザイン |
| メディカル | 医療現場向けの落ち着いたデザイン |

設定は端末に保存され、次回起動時にも引き継がれます。

---

## 5. 注意事項

### データの保存について
- ラウンドデータ（チェック結果・写真・総評）は端末のメモリ上に一時保存されます
- **ブラウザを閉じたり、ページを再読み込みすると、入力中のデータは消去されます**
- ラウンド終了後は速やかにWord出力または共有を行い、記録を保存してください

### 写真について
- 写真に患者の個人情報（氏名・顔・患者番号等）が映り込まないよう注意してください
- 写真データはアプリ内のメモリにのみ保持され、外部サーバーには送信されません

### 音声入力について
- 音声入力を使用する際は、患者情報を音声で発話しないよう注意してください
- 音声認識はデバイスまたはブラウザの機能を使用します

## 6. よくある質問（FAQ）

**Q: オフラインで使えますか？**  
A: 一度アクセスしてホーム画面に追加すると、院内Wi-Fiが届かない環境でも基本操作が可能です。ただし初回のアクセスにはネットワーク接続が必要です。

**Q: 入力中にアプリを閉じてしまいました**  
A: 入力データは保存されていません。最初からラウンドを実施してください。

**Q: 写真が追加できません**  
A: ブラウザへのカメラ・写真ライブラリへのアクセス許可が必要です。端末の設定からブラウザのアクセス許可を確認してください。

**Q: 音声入力ボタンが反応しません**  
A: 一部の端末・ブラウザでは音声入力に対応していません。その場合は手動入力をご利用ください。

**Q: 出力したWordファイルの文字化けがあります**  
A: `.docx` ファイルはMicrosoft Word またはLibreOffice で開いてください。他のソフトウェアでは正常に表示されない場合があります。

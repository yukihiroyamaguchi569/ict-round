# ICN感染対策ラウンドアプリ — 計画書

## 概要
病院のICN（感染管理認定看護師）が感染対策ラウンドで使用するWebアプリ。
各所の写真撮影・音声コメント入力を行い、最終的にレポートをPDF出力する。

## 要件

### 機能要件
1. **ラウンド管理** — 新規ラウンドの開始・終了、日時と担当者名を記録
2. **写真撮影** — カメラで撮影またはギャラリーから選択。場所名を紐付け
3. **音声コメント** — Web Speech API による音声→テキスト変換。手動編集も可
4. **チェックポイント一覧** — 撮影済み写真＋コメントを一覧表示・編集・削除
5. **レポート生成** — 全チェックポイントをまとめたレポートを画面表示し、PDF出力

### 非機能要件
- モバイルファースト（iPad / iPhone Safari で動作）
- オフライン不要（院内Wi-Fi前提）
- バックエンド不要（ローカルストレージ＋クライアント完結）
- 日本語UI

## 技術スタック
- **フレームワーク**: React + TypeScript（Vite）
- **スタイル**: Tailwind CSS
- **写真**: `<input type="file" capture="environment">`
- **音声入力**: Web Speech API (`webkitSpeechRecognition`)
- **PDF生成**: html2canvas + jsPDF
- **データ保存**: localStorage（セッション単位）

## 変更対象
- `package.json` — プロジェクト初期化
- `src/` — アプリ本体
  - `App.tsx` — ルーティング・状態管理
  - `components/RoundStart.tsx` — ラウンド開始画面
  - `components/CheckpointForm.tsx` — 写真＋音声入力フォーム
  - `components/CheckpointList.tsx` — チェックポイント一覧
  - `components/ReportPreview.tsx` — レポートプレビュー＋PDF出力
  - `types.ts` — 型定義
- `index.html` — エントリポイント
- `docs/` — 計画・引き継ぎ・レビュー文書

## 受け入れ条件（Done条件）
1. [ ] ラウンド開始画面で担当者名を入力してラウンドを開始できる
2. [ ] カメラまたはギャラリーから写真を追加できる
3. [ ] 音声入力でコメントをテキスト化できる（手動入力も可）
4. [ ] チェックポイント一覧で写真・コメント・場所名を確認できる
5. [ ] チェックポイントを削除できる
6. [ ] レポートプレビュー画面で全チェックポイントが表示される
7. [ ] PDF出力ボタンでレポートをダウンロードできる
8. [ ] モバイル（iPhone/iPad Safari）で操作できるレスポンシブUI
9. [ ] `npm run build` がエラーなく完了する

## 非対象（スコープ外）
- ユーザー認証・ログイン機能
- サーバーサイド保存・データベース
- 複数ラウンドの履歴管理
- オフライン対応（Service Worker）
- 多言語対応（日本語のみ）

## 画面遷移
```
[ラウンド開始] → [チェックポイント入力] ⇄ [一覧表示] → [レポートプレビュー] → PDF
```
